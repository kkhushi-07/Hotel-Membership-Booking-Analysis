# 🏨 Hotel Membership & Booking Analysis

## 📌 Project Overview

This project demonstrates SQL-based data analysis on a hotel membership system. The dataset contains information about members, memberships, hotel bookings, transactions, entitlements, and renewals.

The objective is to answer business questions using SQL and generate insights that can help understand customer behavior, booking trends, membership performance, and revenue.

---

## 🎯 Objectives

* Analyze hotel booking data using SQL.
* Practice SQL concepts with real-world business problems.
* Generate meaningful business insights.
* Build a professional SQL portfolio project.
* Create an interactive Power BI dashboard based on SQL analysis.

## 📈 Business Questions

Some of the analysis performed includes:

* Display bookings along with member names.
* Calculate booking count for each hotel.
* Calculate booking count for each city.
* Calculate total revenue for each membership product.
* Find average transaction amount.
* Find average revenue per booking.
* Display membership expiry details.
* Analyze active and expired memberships.
* And many more...
