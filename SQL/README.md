# SQL Queries

This folder contains SQL queries written to analyze the **Hotel Membership & Booking** database.

## Objectives

* Practice SQL using real-world business scenarios.
* Strengthen database querying skills.
* Generate business insights from relational data.

## SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions
* INNER JOIN
* LEFT JOIN
* CASE Statements
* Date Functions
* Subqueries

## Tables Used

* Member
* Membership
* Booking
* Hotels
* Transaction
* Entitlements
* Renewals

## Query Format

Each query follows a consistent structure:

* Problem Number
* Question
* SQL Concept
* Tables Used
* Business Purpose
* SQL Solution

This format improves readability and makes the project easier to understand and maintain.
